# Resumo dos estudos
## Jogo da AWS
### Amazon S3
- Serviço de armazenamento.
- Qualquer tipo de arquivo e qualquer metadado que descreve aquele arquivo é chamado de **objeto**.
- Objetos são armazenados em containers S3, chamados **buckets**.
